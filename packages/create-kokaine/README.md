# `@kokaine/create`

Create a Kokaine application backed by npm, Vite, and the managed Koka
toolchain.

```sh
npm create @kokaine my-app
cd my-app
npm install
npm run dev
```

The generated application depends on `@kokaine/core` and uses
`@kokaine/cli` for development, checking, and production builds.

See the [Kokaine repository](https://github.com/Eric-Song-Nop/kokaine) for
the framework documentation and online Playground.
